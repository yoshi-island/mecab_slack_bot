# 実行

```
% git clone https://github.com/yoshi-island/mecab_slack_bot.git
% cd mecab_slack_bot
% pip install --upgrade pip
% pip install -r pip_modules.txt
```

```
% cp password_list_sample.py password_list.py
% vi password_list.py
```
put informations in password_list.py

```
% python run.py
```
